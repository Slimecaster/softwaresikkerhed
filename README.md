## Opgave - Flat File (Svær)
Udarbejdet af Nichlas Damgaard og Amalie Asborg Hansen

### Hvorfor er det smart at bruge en flat_file_db?

En flat file er en simpel datafil, der kan bruges til at gemme information. Det smarte ved en flat file er, at man kan gemme al information i ét enkelt lag i sit program. 
Dette gør det perfekt til simple applikationer, hvor man gemmer sin "database" i samme repository som det dertilhørende program. 
Vi valgte at gemme vores data i vores flat_file_db som struktureret data ved at gemme det i en JSON-fil. 

### Valg af testteknik:
For at finde ud af hvilke funktioner vores database skulle have, valgte vi at bruge testteknikken CRUD(L), fordi vi skal arbejde med data og derfor gerne vil kunne behandle disse.
